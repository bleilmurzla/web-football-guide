# web-football-guide

一个用于归档和发布多个独立 HTML 页面的仓库。这些页面主要围绕足球相关内容整理，以静态网页形式呈现，旨在提供结构化的信息展示与查阅体验。

## 项目简介

本仓库作为独立的 HTML 页面集合，不绑定任何特定域名或网站。所有页面均为纯前端实现，无后端依赖，便于直接通过浏览器打开或托管至任意静态服务。

页面内容聚焦于足球领域相关的资料整理、信息呈现或功能展示，具体形态包括但不限于列表、指南、参考卡片等。

## 目录说明

```
web-football-guide/
├── index.html          # 入口页（页面导航）
├── pages/              # 独立 HTML 页面目录
│   ├── guide-*.html    # 各类指南页面
│   └── ...
├── assets/             # 静态资源（CSS、图片等）
└── README.md           # 本文件
```

- `index.html`：仓库主入口，列出所有已归档页面链接。
- `pages/`：存放所有独立 HTML 页面文件，命名以 `guide-` 前缀区分不同主题。
- `assets/`：页面共用资源，如样式表、图标、图片等。

## 页面归档说明

- 每个页面均为自包含 HTML 文件，可独立运行。
- 页面间通过 `index.html` 组织导航，但无强制依赖关系。
- 归档内容基于通用足球知识领域，不涉及具体赛事、俱乐部或个人的宣传推广。
- 页面更新频率不定，视资料整理进度而定。
